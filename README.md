## 🧠 Core Concepts Covered (So Far)

**Remote Sensing Fundamentals**
- Electromagnetic spectrum, passive vs active sensors
- LEO / GEO / SSO orbits, spatial / spectral / temporal / radiometric resolution
- Sentinel-2 all 13 bands, band combinations, spectral signatures
- TOA vs BOA reflectance, atmospheric correction, cloud masking

**Google Earth Engine (GEE)**
- ImageCollection filtering, cloud masking, NDVI/NDWI/NDBI computation
- Time series charting, median compositing, export to Google Drive
- GEE Python API — `ee.Initialize()`, pulling data into Python pipeline

**Geospatial Python Stack**
- `rasterio` — read/write GeoTIFF, extract bands as numpy arrays
- `geopandas` — vector data, shapefiles, ROI clipping
- `GDAL` — georeferencing, CRS, transform matrix

**ML for Satellite Data**
- Feature engineering from bands (NDVI, NDWI, NDBI, EVI)
- Random Forest + SVM for land cover classification
- Spatial train/test split (vs random split) — why it matters
- CNN basics for satellite imagery — 13-channel input, transfer learning
